## 三、关键发现

### 发现1：Euclidean 的优势随预算增加而放大


| 预算增量  | Euclidean vs Std 自给自足率提升 | Euclidean vs Std 峰值购电降低 |
| ----- | ------------------------ | ----------------------- |
| +10%  | -5.2%                    | +0.4%                   |
| +30%  | -4.1%                    | -0.9%                   |
| +50%  | +3.6%                    | -4.4%                   |
| +100% | **+47.5%**               | **-26.7%**              |


低预算下差距不大，高预算下 Euclidean 优势爆发。

### 发现2：Std 的 Pareto 前沿过早饱和

Std 的成本上限 637,428€，仅比基准高 51%。
Euclidean 的成本上限 1,722,030€，比基准高 308%。

原因：Std 指标 = σ(P_net)，对设备配置不敏感。增加设备后标准差改善有限，NSGA-II 缺乏探索动力。
结论：**Std 方法无法为高预算决策者提供有意义的选择空间。**

### 发现3：Pearson 在中等预算下有竞争力

Pearson 在 +10%~+30% 预算下电网波动最低（433→357 kW），综合表现甚至优于 Euclidean。
但 Pearson 的 Pareto 前沿宽度仅 32.5万€，高预算区间探索不足（75.4万€ 封顶）。

论文叙事：Pearson 方法适合预算有限的场景，但缺乏高预算区间的探索能力。

### 发现4：SSR 高预算下弃光率失控

SSR 在成本水平3（281.9万€）时弃光率高达 61.4%。
原因：SSR 只看"自给自足率"，不区分能质，盲目堆光伏和储能。
对比：Euclidean 在类似成本水平（172.2万€）时弃光率仅 12.2%。

论文叙事：**SSR 方法缺乏能质区分，导致资源配置效率低下。㶲加权方法在提高自给自足率的同时保持了合理的可再生能源消纳。**

### 发现5：Euclidean 弃光率极低

在 +100% 预算下，Euclidean 弃光率仅 0.08%，而自给自足率达 57.0%。
这说明㶲加权引导优化器优先配置储能和CHP，而非盲目堆光伏。

## 五、对审稿人的完整回应

### Q1: 能质系数怎么来的？

A: 卡诺㶲系数，λ_h = 1-T₀/T_h, λ_c = T₀/T_c-1。无人为参数，不同案例自动适应。

### Q2: 值这么小（0.131, 0.064）合理吗？

A: 合理。这反映了热力学事实：1kW 电偏差的㶲损失远大于 1kW 热偏差。优化器因此优先保障电平衡，这正是电网最需要的。实验结果验证了这一点——Euclidean 方案的电网波动最低。

### Q3: 匹配度提高了有什么用？

A: 见 Table X。在 +100% 预算下，Euclidean vs Std：

- 峰值购电 -26.7%（减少配电网投资）
- 年购电量 -29.9%（降低外部依赖）
- 自给自足率 +47.5%（提高系统韧性）
- 弃光率仅 0.08%（高效消纳可再生能源）

### Q4: 为什么不直接优化这些运行指标？

A: 这些指标是离散、非光滑的（涉及 max/min），不适合做 GA 目标。匹配度是连续、可微、多维耦合的代理指标。类比：深度学习用 loss 训练、用 accuracy 评价。

### Q5: Std 方法为什么 Pareto 前沿这么窄？

A: 标准差对设备配置不敏感。σ(P_net) 衡量"波动性"而非"偏差大小"，增加设备后改善有限。这是 Std 方法的本质缺陷——无法有效引导高预算区间的规划决策。
