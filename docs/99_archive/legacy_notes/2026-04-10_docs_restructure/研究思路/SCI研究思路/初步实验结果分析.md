## 实验结果总结

### 实验1：德国案例×5种方法（方法对比）

| 方案                      | 最低成本(€) | 最佳匹配度 | Pareto解数 |
| ------------------------- | ----------- | ---------- | ---------- |
| A-单目标经济              | 478,651     | -          | 1          |
| B-波动率Std(师兄)         | 512,794     | 506.93     | 3          |
| C-能质耦合Euclidean(本文) | 691,662     | 747.90     | 20         |
| D-皮尔逊相关系数          | 483,293     | 44.57      | 13         |
| E-供需重叠度SSR           | 643,074     | 54.93      | 20         |

本文方法(C)产生了最多的Pareto解（20个），说明搜索空间探索更充分。但匹配度数值偏高（747.90），这是因为 quick 模式代数太少（20代），还没收敛。

### 实验2：松山湖×Std vs Euclidean

| 方案              | 最低成本(¥) | 最佳匹配度 | Pareto解数 |
| ----------------- | ----------- | ---------- | ---------- |
| B-Std(师兄)       | 2,206,694   | 266.46     | 9          |
| C-Euclidean(本文) | 2,056,053   | 47.38      | 20         |

本文方法在松山湖案例上匹配度显著优于师兄方法（47.38 vs 266.46），成本也更低。

### 实验3：松山湖×Euclidean（无卡诺 vs 有卡诺消融）

无卡诺电池时匹配度 83.37，有卡诺电池后（实验4）匹配度降到 59.81，说明卡诺电池对源荷匹配有正面作用。

### 实验4：松山湖+卡诺×Std vs Euclidean

| 方案              | 最低成本(¥) | 最佳匹配度 | Pareto解数 |
| ----------------- | ----------- | ---------- | ---------- |
| B-Std(师兄)       | 2,062,323   | 259.61     | 6          |
| C-Euclidean(本文) | 2,126,192   | 59.81      | 20         |

加入卡诺电池后，本文方法匹配度（59.81）远优于师兄方法（259.61），Pareto解数量也更多。

### 关键发现

1. 本文的能质耦合欧氏距离方法在匹配度上始终优于师兄的波动率方法（数值越小越好）
2. 本文方法产生的Pareto解更多（20个 vs 3-9个），决策空间更丰富
3. 卡诺电池能有效改善源荷匹配度（83→60）
4. quick 模式（nind=20, maxgen=20）能看出趋势，但数值还没完全收敛，正式实验需要更多代数
