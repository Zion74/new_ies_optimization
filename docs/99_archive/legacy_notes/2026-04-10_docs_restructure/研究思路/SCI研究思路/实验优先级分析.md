# 实验逻辑链条分析

## 审稿人会问的三个致命问题

按优先级排序，任何一个答不上来论文就挂：

```
Q1: 你的匹配度指标优化出来的方案，跟直接优化经济性的方案，有什么区别？
    → 如果没区别，双目标优化就没意义

Q2: 有区别的话，匹配度更好的方案在实际运行中好在哪？
    → 如果说不出来，匹配度就是个自嗨指标

Q3: 你的匹配度公式（㶲加权欧氏距离）比别人的公式好在哪？
    → 如果没优势，方法创新就不成立
```

注意顺序：Q1和Q2比Q3更致命。Q3是"你比别人好"，Q1Q2是"你这个东西有没有用"。
一个有用但不是最好的指标，比一个最好但没用的指标，更容易发表。

---

## 核心逻辑链（最小必要实验集）

### 第一环：证明"双目标有意义"（回答Q1）

**实验A：单目标 vs 双目标的方案差异**

做法：
- 跑 economic_only（单目标）得到一个最优方案
- 跑 euclidean（双目标）得到 Pareto 前沿，从中选折中解

分析：
- 两个方案的设备配置有什么不同？
- 单目标方案的匹配度是多少？双目标折中解的匹配度改善了多少？成本多花了多少？

如果不做：
- 审稿人会问"为什么不直接最小化成本"，你答不上来
- 这是整篇论文存在的前提

**结论：必做。这是地基。**

---

### 第二环：证明"匹配度好 = 实际运行好"（回答Q2）

**实验B：后验运行指标对比**

做法：
- 从实验A的结果中取2-3个方案（单目标最优、双目标折中、双目标匹配度最优）
- 用完整8760小时跑调度
- 算硬指标：峰值购电、弃光率、自给自足率、电网交互波动

分析：
- 匹配度更好的方案，硬指标是否也更好？
- 具体好多少？（量化）

如果不做：
- 审稿人问"匹配度高了又怎样"，你只能说"理论上应该更好"
- 没有数据支撑，整个论文的实际意义就是空的
- 这是论文最核心的一张表

**结论：必做。这是论文的核心证据。没有这个实验，论文没有说服力。**

---

### 第三环：证明"你的公式比别人好"（回答Q3）

**实验C：多种匹配度方法的Pareto前沿对比**

做法：
- 在同一个案例上跑多种方法（std / euclidean / pearson / ssr）
- 画Pareto前沿对比图

分析：
- euclidean的前沿是否支配其他方法的前沿？
- 各方法推荐的设备配置有什么差异？

如果不做：
- 审稿人问"为什么用欧氏距离不用标准差"，你只能说"理论上更好"
- 方法创新的直接证据缺失

但注意：
- 这个实验的说服力取决于第二环。如果你能证明euclidean的方案运行指标更好（实验B），
  那Pareto前沿对比只是锦上添花
- 如果你只有Pareto前沿对比，没有运行指标对比，审稿人会说
  "你只是换了个目标函数，Pareto前沿当然不一样，但不代表更好"

**结论：必做，但优先级低于实验B。**

---

### 到这里，论文的骨架就立住了：

```
实验A（地基）：双目标 vs 单目标 → 证明匹配度指标有存在的必要
实验B（核心）：后验运行指标 → 证明匹配度好 = 实际运行好
实验C（方法）：多方法Pareto对比 → 证明你的公式比别人好
```

三个实验形成闭环：
  A说"需要匹配度" → C说"我的匹配度最好" → B说"最好的匹配度带来最好的运行效果"

## 扩展实验（在核心链条成立的前提下）

### 扩展1：普适性验证

**实验D：换案例验证（松山湖）**

做法：在松山湖案例上重复实验C（至少跑 std + euclidean）

回答：换一个气候条件、负荷特征完全不同的案例，方法还有效吗？

如果不做：
- 审稿人可能说"你只在一个案例上验证过，普适性存疑"
- 但如果论文篇幅有限，一个案例+充分分析也能接受
- SCI通常期望至少2个案例

**结论：强烈建议做，但不是生死攸关。**

### 扩展2：卡诺电池

**实验E：卡诺电池的价值**

做法：松山湖案例，euclidean方法，有/无卡诺电池对比

回答：卡诺电池对匹配度和经济性的影响

如果不做：
- 论文少一个创新点，但核心创新（匹配度方法）不受影响
- 如果论文定位是"方法论文"，卡诺电池可以不做
- 如果论文定位是"系统论文"（方法+设备），则需要做

**结论：取决于论文定位。如果时间紧，可以先不做。**

### 扩展3：极端场景韧性

**实验F：极端日验证**

做法：选极端天气日，比较不同方案的抗风险能力

回答：匹配度高的方案是否更抗极端天气

如果不做：
- 论文的"韧性"论点缺少直接证据
- 但实验B的运行指标（自给自足率、峰值购电）已经间接说明了韧性
- 极端场景是加分项，不是必要项

**结论：锦上添花。有时间就做，没时间不影响论文成立。**

### 扩展4：能质系数敏感性

**实验G：λ的敏感性分析**

做法：对比 λ=[1,1,1] vs λ=卡诺㶲系数 的优化结果差异

回答：能质系数到底有没有影响？影响多大？

如果不做：
- 审稿人可能问"如果λ取别的值呢"
- 但如果λ有明确的热力学依据（卡诺公式），敏感性分析不是必须的
- 可以在Discussion里简要讨论，不一定要做完整实验

**结论：可以简化为Discussion中的定性讨论，不一定要做完整实验。**

## 优先级排序

```
P0（论文能不能发）：
  实验A：单目标 vs 双目标        → 证明匹配度有存在必要
  实验B：后验运行指标            → 证明匹配度有实际价值
  实验C：多方法Pareto对比        → 证明你的方法最好

P1（论文能不能发好期刊）：
  实验D：松山湖案例验证          → 普适性

P2（论文更丰满）：
  实验E：卡诺电池                → 第二个创新点
  实验F：极端场景                → 韧性证据
  实验G：λ敏感性                 → 参数鲁棒性
```

## 最终建议：先做什么

```
第一步：改 case_config.py（λ改为卡诺公式）
第二步：跑 --exp 1（德国×5种方法）→ 得到实验A+C的原始数据
第三步：写 post_analysis.py → 做实验B（后验运行指标）
第四步：根据实验B的结果决定论文的故事能不能讲通
  - 如果讲得通 → 继续做实验D/E/F
  - 如果讲不通 → 需要回头调整方法或指标
```

第四步是关键决策点。如果实验B显示euclidean方案的运行指标确实优于std，
那整篇论文就立住了，后面的实验都是加分项。
如果实验B显示没有明显差异，那需要重新思考匹配度指标的设计。
