# SCI 结果分析扩充方案

更新时间：2026-04-14

## 1. 这份文档解决什么问题

当前 `Results/服务器结果/full_all_50x100_20260413_134312/` 已经能够支撑论文的四组核心实验结论，但现有分析仍以“最优解结果陈述”为主，距离 SCI 论文中更强的 `Results and discussion` 还差一层“证据链组织”。

本方案的目标不是推翻现有结论，而是把现有结果扩展成更完整的论文型分析结构，使结果部分从“有结论”升级为“有层次、有机制、有兑现验证、有外推边界”。

## 2. 当前 full 结果的优势与短板

### 2.1 已经具备的强信号

- 实验 1 已经证明：德国案例下 `EQD/Euclidean` 与 `Std` 的最低成本接近，但构型偏好不同。
- 实验 2 已经证明：松山湖案例下 `EQD` 会把系统从“纯波动率匹配”引向更强的冷热电耦合配置。
- 实验 3 已经证明：引入卡诺电池后，最低成本仅增加约 `0.62%`，但匹配度改善约 `79.5%`。
- 实验 4 已经证明：在卡诺电池场景下，`EQD` 相对 `Std` 同时实现更低成本和更优匹配，具备“方法-设备协同”证据。

### 2.2 当前分析的主要不足

- 仍以“最低成本解 / 最佳匹配解”叙述为主，缺少对 Pareto 前沿形态的讨论。
- 还没有把“容量变化”进一步上升为“关键设备驱动机制”。
- 还没有充分利用 `scripts/post_analysis.py` 和 `scripts/enhanced_analysis.py` 去做 `8760h` 后验运行兑现分析。
- 对卡诺电池的分析还偏“有无对比”，尚未展开为“价值识别机制”。
- 尚未显式讨论结果的边界条件，例如 `λ` 敏感性、极端工况、负荷增长或价格扰动下的稳定性。

### 2.3 当前数据处理上需要特别注意的一点

当前多组 `Pareto_*.csv` 中存在极高成本尾部解，不能直接把全部 Pareto 点作为论文主图主表的讲述对象。

例如：

- `exp2` 的 `Euclidean` 前沿从 `1.91e6` 一直延伸到 `3.80e12`
- `exp4` 的 `Std` 前沿从 `1.93e6` 延伸到 `5.96e11`
- `exp4` 的 `Euclidean` 前沿从 `1.88e6` 延伸到 `2.83e12`

因此，正式论文中建议先做“代表性解筛选”，再谈 trade-off。否则审稿人很容易质疑：

- 前沿是否包含失真解或不可接受高投资解
- 最佳匹配解是否只是“无限堆设备”得到的结果
- 不同方法之间是否在同一可接受预算区间内比较

## 3. 参考范式：师兄章节和同类论文给出的启发

### 3.1 师兄章节的有效套路

`参考资料/师兄论文章节.pdf` 中，第 4 章结果分析基本遵循以下顺序：

1. `Pareto 权衡关系`
2. `容量配置结果分析`
3. `运行阶段典型日调度分析`
4. `负荷增长场景下的长期效益分析`
5. `研究总结`

这个结构的优点是：

- 先讲“值不值得权衡”
- 再讲“哪些设备导致了这种权衡”
- 然后讲“这种规划在运行层面如何体现”
- 最后讲“这种差异在长期场景下是否有价值”

这正是你当前结果分析最适合借鉴的主骨架。

### 3.2 同类论文常见的高质量分析模块

结合近年 IES 规划文献，类似研究通常不会只停留在 Pareto 图上，而会继续补以下几类证据：

- `能质/㶲视角的目标合理性验证`
- `容量配置差异与关键设备驱动机制`
- `全年运行指标兑现`
- `可再生消纳/自给率/外购电等运行收益`
- `不确定性或场景扰动下的稳健性`
- `长期负荷增长或生命周期收益分析`

对本研究最直接有用的启发来自以下文献：

- Fan et al., 2021, *Review on Coordinated Planning of Source-Network-Load-Storage for Integrated Energy Systems*：强调 IES 规划不能只看单一静态最优值，还要考虑负荷增长、多阶段决策和不确定性边界。
- Liu et al., 2020, *Multi-objective planning for integrated energy systems considering both exergy efficiency and economy*：说明“经济性 + 能质/㶲指标”的双目标规划，必须同时讨论目标合理性和规划结果。
- Wang et al., 2021, *Integrated Energy System Planning Based on Life Cycle and Emergy Theory*：除了成本，还会补充可再生能源消纳、生命周期收益等结果分析维度。
- 近年关于 `source-load forecasting uncertainty`、`reliability and low-carbon joint optimization` 的 IES 规划文章普遍会补做场景扰动、风险或稳健性分析，而不是只用静态工况验证方法优劣。

## 4. 面向你当前论文的结果分析新方案

建议把 `Results and discussion` 扩成六个分析模块。核心原则是：

`先做前沿筛选 -> 再讲容量机制 -> 再讲运行兑现 -> 再讲设备价值 -> 最后讲边界条件`

### 模块 A：Pareto 前沿形态与“可接受预算区间”分析

目标：

- 不是只比较最低成本点，而是回答“在合理预算范围内，EQD 能换来什么”

建议做法：

- 对每组实验的 Pareto 解先做代表性筛选
- 只保留 `最低成本解`、`膝点解`、`高匹配但可接受预算解` 三类代表方案
- 对存在超高成本尾部的前沿，主图只展示“可接受预算区间”，附录再给完整图

推荐筛选规则：

- 成本不超过本方法最低成本的 `+20%` 或 `+30%`
- 或者按 `economic_only` 基线做 `+0% / +10% / +30% / +50% / +100%` 预算对齐
- 若要统一所有方法，优先使用“预算对齐”而不是“各自最低成本归一化”

推荐指标：

- Pareto 前沿宽度
- 膝点位置
- 单位匹配改善的边际成本
- 在预算约束下的最优匹配收益

你当前结果中最值得在这里写的点：

- `exp1` 中 `EQD` 与 `Std` 的最低成本仅差约 `0.48%`，但设备结构明显不同。
- `exp2/exp4` 中原始 Pareto 存在超高成本尾部，因此必须使用“预算对齐后前沿”来讲故事。

推荐图表：

- 图 A1：裁剪后的 Pareto 前沿图
- 图 A2：各方法在预算增量下的最优匹配收益图
- 表 A1：代表性解筛选表

### 模块 B：关键设备驱动机制分析

目标：

- 把“容量配置不同”升级为“哪些设备在驱动匹配度变化”

建议做法：

- 不再只罗列全部设备，而是聚焦 `GT / HP / EC / AC / ES / HS / CS / CB`
- 对每组实验的代表性解做容量差分分析
- 强调从“热驱动路径”到“电驱动 + 耦合储能路径”的迁移

当前结果里已经很清楚的几个信号：

- `exp2` 中，`EQD` 相比 `Std`：`GT` 从 `0` 增至 `69 kW`，`AC` 从 `125.5` 增至 `186.1 kW`，`HS` 从 `406.4` 增至 `446.2 kW`，`CS` 从 `2978.3` 降至 `2902.2 kW`。
- `exp4` 中，`EQD` 相比 `Std`：`GT` 从 `83.2` 降至 `2.0 kW`，`EC` 从 `2014.1` 升至 `2212.2 kW`，`AC` 从 `346.4` 降至 `79.5 kW`，`CB_Power` 从 `120.7` 升至 `238.1 kW`，`CB_Capacity` 从 `467.7` 升至 `1211.8 kWh`。

这说明：

- 无卡诺时，`EQD` 通过增强热电联供与冷热耦合来改善匹配。
- 有卡诺时，`EQD` 更倾向于“电驱动设备 + 卡诺电池”协同，而不是继续放大传统 `GT + AC` 路径。

推荐图表：

- 图 B1：代表性解容量配置雷达图或分组柱状图
- 图 B2：随成本增加的容量演化图
- 表 B1：关键设备增减幅度表

### 模块 C：8760h 后验运行兑现分析

目标：

- 回答“规划层的匹配改进，是否真的在全年运行中兑现”

这部分建议直接调用：

- `scripts/post_analysis.py`
- `scripts/enhanced_analysis.py`

推荐运行指标：

- 峰值购电 `peak_grid_kW`
- 年购电量 `annual_grid_purchase_MWh`
- 自给自足率 `self_sufficiency_%`
- 电网交换波动 `grid_volatility_kW`
- 弃电或溢流指标

建议论文里的核心表述方式：

- 不说“EQD 在 Pareto 图上更好”，而说“EQD 对应的规划方案在全年运行中表现为更低峰值外购电 / 更高自给率 / 更低电网波动”

推荐图表：

- 图 C1：预算对齐下的全年运行指标对比
- 表 C1：代表性解 `8760h` 后验运行指标表

### 模块 D：典型日调度机制分析

目标：

- 把运行兑现继续解释到“能流耦合机理”

建议做法：

- 选取 `1` 个可再生波动大的典型日
- 选取 `1` 个冷负荷高峰典型日
- 比较 `Std vs EQD`，以及 `无卡诺 vs 有卡诺`

推荐展示内容：

- 电调度图：看外购电削峰填谷、储能充放电、GT 参与程度
- 热调度图：看热泵、GT 余热、热储能协同
- 冷调度图：看 `EC/AC/CS` 的角色重分配

这一模块在叙事上的作用是：

- 解释“为什么容量差异会转化为运行收益”
- 让“EQD 识别卡诺电池价值”从结果结论变成机制结论

推荐图表：

- 图 D1：典型日电调度对比
- 图 D2：典型日热/冷调度对比

### 模块 E：卡诺电池价值与方法-设备协同分析

目标：

- 把实验 3 和实验 4 从简单对比升级为“第二贡献闭环”

建议拆成两个子问题：

1. `exp3`：卡诺电池本身是否有价值
2. `exp4`：这种价值是否依赖于匹配度指标是否足够合理

你当前结果可以直接支撑的写法：

- `exp3`：加入卡诺电池后，最低成本仅增加约 `0.62%`，匹配度改善约 `79.5%`，说明卡诺电池具有高匹配收益、低经济代价。
- `exp4`：在有卡诺场景下，`EQD` 相比 `Std` 成本下降约 `2.51%`，同时匹配更优，说明 `EQD` 能更好识别卡诺电池的系统价值。

推荐进一步增加的指标：

- 单位卡诺容量带来的匹配改善
- 单位卡诺容量带来的成本变化
- `CB_Power / CB_Capacity` 与 `EC/AC/GT` 的协同变化

- 图 E1：无卡诺 vs 有卡诺的成本-匹配收益图
- 图 E2：有卡诺条件下 `Std vs EQD` 的价值识别图
- 表 E1：卡诺电池价值汇总表

### 模块 F：边界条件、敏感性与扩展场景分析

目标：

- 避免论文停留在“单一工况下的漂亮结果”

优先级建议：

#### F1. λ 敏感性或消融实验

直接对应：

- `scripts/lambda_sensitivity.py`

建议回答的问题：

- 若热/冷能质权重减弱，`EQD` 优势是否减弱
- 若令 `λ_e = λ_h = λ_c = 1`，是否会退化为近似“无能质差异”的几何距离

#### F2. 价格与负荷扰动稳健性

建议场景：

- 电价 `±20%`
- 气价 `±20%`
- 夏季冷负荷峰值上浮 `10%~20%`

建议输出：

- 方法排序是否改变
- 成本 regret 是否扩大
- 卡诺电池是否仍保有价值

#### F3. 负荷增长长期效益

这是最直接借鉴师兄章节的模块。

如果不想把硕士论文的长期规划讲太深，可把它作为“Discussion / Appendix”中的补强实验，而不是正文主结论。

## 6. 推荐的正文组织方式

建议把论文 `Results and discussion` 中每个实验小节都统一为“四段式”，这样行文最稳：

1. `结果层`
   给出该实验最核心的数字结论。
2. `结构层`
   解释关键设备容量是怎么变化的。
3. `机制层`
   用典型日或全年运行指标说明这种容量差异如何兑现。
4. `意义层`
   说明这一实验支撑全文中的哪条主线。

具体到你当前论文，可以写成：

- `6.1` 德国案例：证明 `EQD` 改变构型偏好，而不仅是改目标函数值。
- `6.2` 全年运行兑现：证明规划层优势可落到运行层。
- `6.3` 松山湖案例：证明跨案例普适性。
- `6.4` 卡诺电池消融：证明设备本身的配置价值。
- `6.5` 有卡诺条件下 `Std vs EQD`：证明方法与设备存在协同增益。
- `6.6` 敏感性与边界：说明结果不是偶然。

## 7. 执行优先级

### 必做

- 做代表性解筛选，避免直接用全 Pareto 尾部解讲故事
- 做 `8760h` 后验运行指标
- 把实验 3 和实验 4 联合写成“卡诺电池价值识别闭环”

### 强烈建议做

- 做 1~2 个典型日调度图
- 做 `λ` 敏感性或消融实验

### 有余力再做

- 负荷增长长期效益
- 更系统的价格/极端气候稳健性分析

## 8. 对你这篇论文最稳妥的结论升级方式

基于这个方案，整篇论文的结果分析部分可以从现在的三条主线：

- 方法改变构型偏好
- 卡诺电池改善匹配
- 方法与设备存在协同

升级为更完整的五条主线：

1. `EQD` 改变了 Pareto 权衡关系，而不是仅改变一个指标数值。
2. `EQD` 会系统性改变关键设备配置路径。
3. 这种规划差异能在 `8760h` 运行中兑现为外购电、自给率与波动性的改进。
4. 卡诺电池本身具有高匹配收益、低成本代价。
5. `EQD` 能比 `Std` 更好地识别卡诺电池这类电热耦合设备的配置价值，且这种优势在边界扰动下仍可保持。
