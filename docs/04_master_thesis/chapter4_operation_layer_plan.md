# 硕士论文第 4 章：运行层计划

更新时间：2026-04-10

## 1. 本章使命

第 4 章的任务不是重新做一篇“预测论文”，而是把前两章的规划与设备层结论放到真实运行语境下检验，回答：

> 如果负荷和可再生出力只能提前预测，`Std`、`EQD`、`EQD+CB` 这些方案谁在日前调度与实时运行中更稳？

## 2. 关键研究问题

1. 多能负荷（电 / 热 / 冷）能否构建一个足够稳定的预测输入层？
2. 预测误差会怎样影响日前调度方案与实时运行指标？
3. 规划层中更优的匹配度方案，是否在运行层也表现出更小的退化？
4. 卡诺电池提供的电热耦合灵活性，是否会转化成更强的运行鲁棒性？

## 3. 建议实验包

### O1：多能负荷预测实验

- 输入：历史电 / 热 / 冷负荷、天气、时间特征。
- 输出：日前 24h 多能负荷预测值。
- 评价：MAE、RMSE、MAPE、nRMSE、按季节分解误差。

### O2：日前调度与实时评估实验

- Step 1：用 O1 的预测结果做日前调度。
- Step 2：用真实负荷回放执行结果。
- Step 3：统计成本增量、峰值购电增量、弃能变化、供能缺口。

### O3：方案间鲁棒性对比

- 对比对象：`Std`、`EQD`、`EQD+CB`。
- 关注指标：运行成本退化、峰值购电退化、外部购能增加、调度失败风险。
- 目标：把前两章的“规划优势”转译为“运行抗扰动能力”。

### O4：典型季节 / 极端日剖面分析

- 场景：夏季高冷负荷日、冬季高热负荷日、阴雨低可再生场景、电价冲击场景。
- 目的：解释不同方案为何会在运行层表现出不同退化轨迹。

## 4. 指标建议

| 指标 | 含义 |
|---|---|
| 运行成本增量 | 预测误差造成的额外成本 |
| 峰值购电增量 | 电网友好性是否恶化 |
| 年购电量增量 | 外部依赖是否增加 |
| 供需失衡时数 / 失衡量 | 调度失败或约束紧张程度 |
| 弃能增量 | 可再生消纳恶化程度 |
| 卡诺电池利用率 | 设备层灵活性是否在运行层被兑现 |

## 6. 与前两章的闭环关系

- 第 2 章证明 `EQD` 在规划层更好。
- 第 3 章证明 `EQD` 更能识别卡诺电池价值。
- 第 4 章要继续证明：这些“规划阶段的好方案”在运行层不是纸上谈兵，而是退化更小、鲁棒性更强。
