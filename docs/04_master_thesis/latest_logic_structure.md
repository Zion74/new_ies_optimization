# 硕士论文：最新逻辑结构

更新时间：2026-04-10

## 1. 定位

硕士论文承担“**把会议论文与 SCI 论文纳入一个更大的研究闭环**”的任务，因此它不能只是一篇长一点的 SCI，而要有清晰的章节递进：

> 第 2 章解决规划层方法问题；第 3 章解决设备层扩展问题；第 4 章解决运行层落地与鲁棒性问题。

## 2. 总体章节逻辑

### 第 1 章：绪论

- 问题背景
- 国内外研究现状
- 三层研究内容与技术路线
- 论文组织结构

### 第 2 章：规划层

主问题：**怎样量化源荷匹配，才能让容量规划更合理？**

- IEMI / EQD 的关系与方法定位
- 两案例、多方法实验
- 8760h 后验运行指标与 λ 敏感性
- 对应会议论文核心内容，也是 SCI 的第一条主线

### 第 3 章：设备层

主问题：**卡诺电池这类电热耦合储能值不值得配，匹配度指标能否识别它的价值？**

- 卡诺电池建模与决策变量扩展
- 无卡诺 vs 有卡诺
- 有卡诺条件下 `Std vs EQD`
- 对应 SCI 的第二条主线

### 第 4 章：运行层

主问题：**规划层和设备层得到的优选方案，在预测误差和实时运行扰动下是否仍然稳健？**

- 多能负荷预测
- 日前调度 / 实时评估
- 鲁棒性验证
- 这是硕士论文相对会议和 SCI 新增的闭环章节

### 第 5 章：结论与展望

- 三层研究结论分别回答了什么问题
- 各层之间如何形成闭环
- 后续可扩展方向

## 3. 与会议论文 / SCI 论文的关系

- **会议论文**：可以看作硕士论文第 2 章中“先把核心规划层概念讲硬”的前导成果。
- **SCI 论文**：可以看作硕士论文第 2 章 + 第 3 章的正式论文化凝练版本。
- **硕士论文新增价值**：第 4 章运行层，把“规划好不好”落到“预测误差下还能不能用”。

## 4. 写作边界

- 第 2 章不要被卡诺电池细节带偏，仍然以规划层方法为主。
- 第 3 章不要重新讲一遍第 2 章全部理论，只承接必要方法基础。
- 第 4 章不要重新发明新的规划指标，而是验证前两章方案在运行层的可兑现性与鲁棒性。

## 5. 当前状态

- 第 2 章：逻辑最成熟，代码与实验基本齐备。
- 第 3 章：模型和实验边界已明确，可继续深化卡诺电池价值分析。
- 第 4 章：逻辑框架已确定，部分韧性脚本已存在，但完整预测与日前调度模块仍需新增。
