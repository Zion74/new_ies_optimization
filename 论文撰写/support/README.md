# 支持材料（`论文撰写/support`）

本目录存放写作支持、审稿备忘和会议辅助材料，不直接承担“最新研究逻辑说明”的权威角色。

## 当前内容

- `文献支撑总结.md`
- `审稿人问答速查表.md`
- `scientific-writer.md`
- `conference_agents.md`
- `conference_guides/`
- `conference_word/`
- `sci_structure_note.md`
- `sci_review_note.md`

## 使用原则

- 需要看最新研究主线：去 `docs/`
- 需要改正式稿件：去 `论文撰写/会议/` 或 `论文撰写/paper/`
- 需要查历史审查、写作提示、会议辅助材料：留在这里
