# 论文撰写目录说明

本目录现在只承担**稿件源码与写作支持材料**的组织功能；最新研究架构、三层映射、实验 / 图表 / 代码边界统一以 `docs/` 为准。

## 当前结构

- `会议/`：会议稿正式源码、图表、数据与 README。
- `paper/`：SCI 稿正式源码、图表、参考文献与 README。
- `reports/`：给导师或阶段汇报材料。
- `support/`：写作支持、审稿备忘、会议辅助材料、历史审查记录。
- `deep research/`：深度文献检索过程材料。
- `相似paper/`：相似论文对照材料。

## 先看哪里

- 看最新研究架构：`docs/README.md`
- 改会议稿源码：`论文撰写/会议/`
- 改 SCI 稿源码：`论文撰写/paper/`
- 找支持材料：`论文撰写/support/`
- 找汇报材料：`论文撰写/reports/`

## 纪律

- 不再把“最新版结构说明”“实验总清单”“章节规划”直接丢回本目录根部。
- 结构类材料进入 `docs/`，历史结构稿进入 `docs/99_archive/`。
