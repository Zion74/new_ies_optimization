## 研究主线

你的论文核心论点是：**"用能质系数加权的欧氏距离来度量源荷匹配度，比现有方法更合理"**。要证明这个论点，需要三步：

**第一步：提出方法** → 为什么欧氏距离+能质系数是合理的（理论推导） 

**第二步：对比实验** → 在多个案例下，方案C比A/B/D/E好在哪（数值实验） 

**第三步：分析差异** → 为什么好，好多少，什么场景下优势最大（结果分析）

## 需要的实验数据

每个案例（德国+松山湖）× 每种方法（A/B/C/D/E）= 10组实验，每组产出：

**1. Pareto前沿数据**（最核心）

- 每组实验的Pareto最优解集：`ObjV`（成本, 匹配度）和 `Phen`（9个设备容量）
- 这是对比的基础——同样的成本下，哪种方法的匹配度更好

**2. 最优方案的设备配置**

- 从Pareto前沿中选取"折中解"（比如TOPSIS法），得到每种方法推荐的设备容量组合
- 对比不同方法推荐的配置差异

**3. 最优方案的运行调度结果**

- 用选定的设备配置，跑完整8760小时的运行模拟
- 提取逐时数据：各设备出力、购网电量、弃风弃光量、储能充放电

**4. 源荷匹配度的逐时分解**

- 电/热/冷三个维度的净负荷时序曲线
- 用5种方法分别计算匹配度，看差异在哪些时段最大

## 分析逻辑（论文章节对应）

### 分析1：Pareto前沿对比（证明方案C找到了更好的解）

把5种方法的Pareto前沿画在同一张图上（横轴成本，纵轴匹配度）。你要证明的是：方案C的Pareto前沿更靠近原点（同等成本下匹配度更低，或同等匹配度下成本更低）。

两个案例分别画一张图，如果两张图都显示方案C优于其他方法，就证明了普适性。

### 分析2：设备配置对比（证明方案C给出了更合理的配置）

做一张表，对比5种方法推荐的设备容量：

| 设备 | 方案A | 方案B | 方案C | 方案D | 方案E |
| ---- | ----- | ----- | ----- | ----- | ----- |
|      |       |       |       |       |       |

预期发现：

- 方案A（纯经济）：CHP偏大（发电省钱），储能偏小
- 方案B（波动率）：各设备均匀，但电热冷之间没有耦合考虑
- 方案C（能质耦合）：会更合理地平衡电热冷设备比例，因为考虑了能质差异
- 方案D（相关系数）：可能配置极端，因为只看趋势不看大小
- 方案E（自给率）：倾向于过度配置可再生能源

### 分析3：运行特性对比（证明方案C的配置运行效果更好）

用每种方法的推荐配置跑全年调度，对比：

- 年购网电量（越少越好）
- 弃风弃光率（越低越好）
- 外部补热/补冷量（越少越好）
- 电网最大需量（越小越好）
- 系统综合能效

### 分析4：能质系数的作用（证明考虑能质差异是必要的）

这是你的核心创新点的直接验证。对比方案B（不考虑能质）和方案C（考虑能质）：

- 在德国案例（热主导）：λ_h=0.6 使得热匹配被适当降权，避免过度配置供热设备
- 在松山湖案例（冷主导）：λ_c=0.5 使得冷匹配被适当降权，避免过度配置制冷设备
- 如果去掉能质系数（令λ_e=λ_h=λ_c=1），结果会变差多少？→ 做一组消融实验

### 分析5（可选）：卡诺电池的影响

如果加入卡诺电池，对比有/无卡诺电池时：

- 源荷匹配度改善了多少
- 系统成本变化
- 卡诺电池的余热回收对热匹配度的贡献

## 实验运行顺序建议



```
1. 德国案例 × 5种方法（full模式，nind=50, maxgen=100）
2. 松山湖案例 × 5种方法（同上）
3. 从每组Pareto前沿选折中解，跑全年8760h详细调度
4. 消融实验：λ_e=λ_h=λ_c=1 的方案C'（证明能质系数的必要性）
5.（可选）松山湖+卡诺电池 × 方案C
```

第1、2步是最耗时的（各1-3小时），后面的分析都是基于这些结果做后处理。







## 两个创新点的关系

- 创新点1：能质耦合欧氏距离匹配度（方法创新）
- 创新点2：含卡诺电池的IES容量优化（模型创新）

这两个是**递进关系**，不是并列关系。创新点1是基础，创新点2是在1的基础上扩展。

## 实验设计的最小必要集

你需要的实验其实是三组，不是十组：

**第一组：方法对比（证明创新点1）**

在**一个案例**上做5种方法的完整对比就够了。德国案例已经有结果了，如果跑得好，松山湖案例作为补充验证。

这组实验回答：方案C比A/B/D/E好。

**第二组：卡诺电池效果（证明创新点2）**

在松山湖案例上，用方案C（你的最优方法），对比：

- 无卡诺电池的方案C
- 有卡诺电池的方案C

这组实验回答：加入卡诺电池后，源荷匹配度和经济性如何变化。

**第三组：能质系数对卡诺电池的影响（把两个创新点串起来）**

含卡诺电池的松山湖案例上，对比：

- 方案B + 卡诺电池（不考虑能质）
- 方案C + 卡诺电池（考虑能质）

这组实验回答：在含卡诺电池的系统中，能质耦合匹配度是否依然优于传统方法。这是把两个创新点绑在一起的关键实验。

## 所以你不能省的是

5种方法的基础对比（第一组）不能省，因为这是创新点1的直接证据。审稿人会问"你的方法比别人好在哪"，没有对比实验就没有说服力。

## 可以省的是

- 不需要两个案例都跑全部5种方法。德国案例跑5种方法做完整对比，松山湖案例只跑方案B和方案C做简要验证就行
- 卡诺电池实验只需要在松山湖案例上做，不需要德国案例也做一遍

## 最终实验清单

```
实验1: 德国案例 × 5种方法          → 证明方案C最优（创新点1主实验）
实验2: 松山湖案例 × 方案B + 方案C  → 验证方案C在不同场景下的普适性
实验3: 松山湖案例 × 方案C（无卡诺） vs 方案C（有卡诺）  → 证明卡诺电池的价值（创新点2）
实验4: 松山湖案例 × 方案B（有卡诺） vs 方案C（有卡诺）  → 串联两个创新点
```

四组实验，逻辑链完整，工作量也可控。
