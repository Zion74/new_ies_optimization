## 1. 负荷特征：热主导 vs 冷主导

| 指标       | 德国        | 松山湖      | 差异                              |
| ---------- | ----------- | ----------- | --------------------------------- |
| 年电负荷   | 1758万kWh   | 166万kWh    | 德国是松山湖的10倍                |
| 年热负荷   | 512万kWh    | 59万kWh     | 德国热负荷大（供暖）              |
| 年冷负荷   | 233万kWh    | 291万kWh    | 松山湖冷负荷反而更大              |
| 冷/电比    | 0.13        | 1.75        | 松山湖冷负荷远超电负荷            |
| 冷/热比    | 0.46        | 4.96        | 核心差异：德国热>冷，松山湖冷>>热 |
| 冷负荷峰值 | 356 kW      | 5797 kW     | 松山湖峰值是德国的16倍            |
| 温度范围   | -9~32°C     | 6~38°C      | 东莞全年温暖，无采暖需求          |
| 风速       | 均值4.9 m/s | 均值1.8 m/s | 东莞城区风资源差                  |

这意味着两个案例下，优化算法会给出完全不同的设备配置方向：德国案例倾向于配大CHP+热泵来满足供暖，松山湖倾向于配大电制冷+蓄冷来满足制冷。

## 2. 经济参数：两部制电价 vs 统一电价

德国案例的电价结构很特殊——基础电价极低（0.0025 €/kWh），但有高额容量费（114.29 €/(kW·a)）。这意味着**削减电网最大需量**比减少用电量更重要，所以德国案例会倾向于配大储能来削峰。

松山湖是统一电价0.6746元/kWh，没有容量费。电价本身较高，所以**自发电（CHP+光伏）的经济性更好**，减少购电量就是省钱。

另外松山湖没有风电（上界=0），因为东莞城区风速太低，配风机不经济。

## 3. 设备参数：效率差异反映气候和设备代际

- CHP发电效率：松山湖0.35 > 德国0.33（CAT G3512E是较新机型）
- CHP供热效率：松山湖0.45 < 德国0.50（松山湖不需要那么多热，余热回收设计不同）
- 电制冷COP：松山湖3.5 > 德国2.87（松山湖用的是高效离心机组，且冷水温差更大）
- 电热泵COP：松山湖4.0 < 德国4.44（德国温带气候热泵效率更高）

## 对你研究的意义

这两个案例形成了很好的对比实验设计：

- 德国案例下，能质系数中 λ_h=0.6 的热权重很重要，因为热负荷大，源荷匹配度主要由电-热耦合决定
- 松山湖案例下，λ_c=0.5 的冷权重更关键，因为冷负荷主导，源荷匹配度主要由电-冷耦合决定

如果你的能质耦合欧氏距离方法（方案C）在两种截然不同的负荷结构下都优于其他方法，就能证明这个方法的**普适性**，而不是只在某一种特定场景下有效。这是论文中很有说服力的论证。
