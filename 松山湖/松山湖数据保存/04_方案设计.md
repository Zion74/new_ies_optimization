## 3. 总体思路

来源：PDF 3.3.1

核心原则："**温度对口，梯级利用**"

将燃烧产生的能量根据温度从高到低分成：

| 温段 | 用途 |
|------|------|
| 高温段 | 发电（燃气内燃机） |
| 中温段 | 驱动吸收式制冷机（溴化锂） |
| 低温段 | 生产生活热水 |
| 排放段 | 尽量降低排放温度，减少损失 |

互补与协同策略：

- **电源互补**：内燃机 + 光伏协同互补运行，提高系统可靠性
- **冷源协同**：吸收式制冷 + 压缩式（电）制冷协同互补运行
- **蓄能调控**：利用东莞日益拉大的峰谷电价差，低谷电蓄冷
- **余热除湿**：利用余热进行除湿，提高空调系统效率

## 4. 方案要点

来源：PDF 3.3.2

1. **能的梯级利用**：根据"温度对口，梯级利用"原则
2. **多能互补与主动蓄能调控**：太阳能 + 天然气多种能源输入，结合相变蓄冷单元，解决电冷热输出比例可调
3. **模块化集成**：撬装模块化集成，标准化、模块化、智能化设计

### 5.1 工艺方案逻辑（图 8 上方框图）

```
[能量输入]      → [能量转换]           → [能量储存]   → [使用]
太阳能          → PV (光伏)           ─┐
天然气          → 燃气内燃机          ─┼→ 蓄冷       → 室内供冷
谷电            → 吸收式制冷/除湿      ─┘            → 照明
                → 水冷多联机                          → 实验用电
                                                     → 充电桩
                                                     → 其他用电
```

### 5.2 工艺方案物理流程（图 8 下方流程图）

```
天然气 → 调压装置 → 燃气内燃机
                    ├─ 烟气 450°C → 吸收式制冷机组 → 冷冻水 7°C → 蓄冷 → 供冷
                    │                                              ↑
                    │                烟气 170°C                    │
                    │                                              │
                    ├─ 缸套水 → 缸套水换热器 → 中温缓存储罐         │
                    │                                              │
                    │                  烟气 100°C → 烟气-热水换热器 → 吸收式除湿机组 → 除湿
                    │                                              │
                    │                                              → 生活热水
                    │
光伏发电 → 供电
压缩式制冷机组 ← 供电（电网或光伏）→ 冷冻水 7°C → 供冷
电网 ⇄ 供电
```

### 5.3 关键温度参数

| 介质 / 流程 | 温度 |
|-------------|------|
| 烟气进吸收式制冷机 | 450°C |
| 烟气出吸收式制冷机 | 170°C |
| 烟气进烟气-热水换热器 | 170°C |
| 烟气最终排放温度 | 100°C |
| 吸收式制冷出冷冻水 | 7°C |
| 压缩式制冷出冷冻水 | 7°C |

## 6. 推荐技术

来源：PDF 4.2

### 6.1 模块化冷热电分布式能源技术

| 项目 | 数值 |
|------|------|
| 建安费下降 | 约 60%（较传统） |
| 项目单位投资 | < 8000 元/kW |
| 投资回收期 | 缩短 3 ~ 5 年 |

### 6.2 蓄冷技术

| 项目 | 数值 |
|------|------|
| 蓄冷介质 | 共晶盐相变蓄冷 |
| 普通空调机组出水温度 | 4.5°C（蓄冷工况） |
| 传统冰蓄冷出水温度 | -5°C |
| 优势 | 不需要低温冷水机组，传统机组可用，能效高、耗电少 |

### 6.3 光伏发电技术

通用光伏技术，光生伏特效应，安全可靠、无噪声、低污染、无需消耗燃料。

### 6.4 智慧能源管理与调度技术

整合电、热、冷、汽、水等能源数据，提供：
- 数据存储、实时监控、可视化管理
- 数据分析、风险控制、能效分析、能源规划

技术基础：云计算、物联网、区域性能源信息公共服务平台。

## 7. 项目装机要点

来源：PDF 4.3

设计要点：

1) **运行时间约束**：
   - 负荷需求时间：8:00 - 19:00
   - 系统供能时间不超：3300 小时/年
   - 冷负荷需求期：4 ~ 11 月
   - 供冷时间不超：2800 小时/年

2) **年总量上限**：
   - 年累计供电量不超：84 万 kWh
   - 年累计供冷量不超：291 万 kWh（83 万 RT）

3) **负荷分布特征**（来自延时曲线）：
   - 全年电负荷量占比最高的负荷段：400 ~ 500 kW
   - 全年冷负荷量占比最高的负荷段：2000 ~ 2500 kW（568 ~ 710 RT）

4) **蓄冷时段**：
   - 夜间低谷蓄冷时间：0:00 - 8:00
   - 每天蓄冷 8 小时

### 7.1 装机规模决策依据

| 设备 | 装机 | 决策依据 |
|------|------|----------|
| 燃气内燃机三联供 | 500 kW | 对应电负荷最高频段 400~500 kW |
| 溴化锂（吸收式制冷） | 550 kW | 用内燃机余热驱动 |
| 压缩式电制冷 | 1500 kW | 补充制冷，覆盖中负荷段 |
| 蓄冷 | 1000 RT（约 3517 kW） | 削峰、利用低谷电 |
| 光伏 | 500 kW（3000 m²） | 屋顶可用面积约束 |
