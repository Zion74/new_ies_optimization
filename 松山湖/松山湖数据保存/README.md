# 松山湖示范方案数据保存

本文件夹保存从《"松山湖国际创新创业社区"MW级多能互补分布式能源示范方案（项目一期）》PDF 中提取的所有结构化数据，供后续优化模型、论文写作、数据校核使用。

## 使用规则

1. 所有数据均直接来自 PDF。文字数据来自正文段落，表格数据来自原表格，图像数值来自人工目读（已在文档中标注出处）。
2. 凡是 PDF 没有给出原始数字、需要用户报读图像的，统一记录在 `99_待补充图像数据.md`。
3. 修改本文件夹内任何数据时，必须同时更新本 README 的"最近更新"时间戳。
