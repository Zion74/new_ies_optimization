## 5. 设备效率参数（用于优化模型）

来源：Codex 通过 CAT 官方参数 + PDF 表4反算 + 行业经验值综合确定。PDF 原文未直接给出以下任何效率值。
已由用户确认（2026-04-16）。

| 设备 | 效率参数 | 建议取值 | 来源与说明 |
|------|----------|----------|----------|
| CAT G3512E | 发电效率 | **42.30%** | CAT 官方技术规格"Maximum Electrical Efficiency"；PDF 表4 5月数据反算（耗气1.55万Nm³→发电6.6万kWh）得 ~42.5%，吻合 |
| CAT G3512E | 余热回收效率 | **40-45%** | PDF 3.1 要求综合能源利用率 ≥ 80%；80%-42% = 38-43%，加上管道损失取 40-45% |
| 溴化锂 YRX443 | COP | **0.70-0.75** | 单效烟气热水型溴化锂（驱动热源 ~90°C），行业典型值 |
| 压缩式电制冷 1500kW | COP | **5.0-6.5** | 水冷螺杆 5.0-5.5；磁悬浮/高效离心 5.5-6.5；建议折中取 **5.5** |
| 相变蓄冷 | 充放效率 | 0.90 / 0.90 | 共晶盐工程值 |
| 光伏 500kW | 标准效率 | ~17% | 当前主流单晶硅组件 |
| 天然气 | 低位热值 | **~10.0 kWh/Nm³** | PDF 表4反算 10.06；国标典型值 9.89（36 MJ/Nm³ → 10.0 kWh/Nm³） |

⚠️ 注意：当前 `case_config.py` 中部分参数与上表有差异：
- `gt_eta_e = 0.35`（代码）vs 42.30%（CAT 官方）→ 偏低，需更新
- `ec_cop = 3.5`（代码）vs 5.0-6.5（核查值）→ 偏低，需更新
- `gas_price = 0.45 元/kWh`（代码，即 4.5/10）vs 4.5/9.89 = 0.455 → 基本一致

这些差异将在数据重建时一并修正。
