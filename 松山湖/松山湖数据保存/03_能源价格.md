## 3. 天然气价格

| 项目 | 数值 |
|------|------|
| 供气方 | 东莞新奥燃气集团 |
| 接驳点距 A4 栋距离 | < 1 km |
| 管道燃气压力 | 0.35 MPa |
| 管道天然气价格 | 4.5 元/Nm³ |
| 备注 | 根据用量情况，可协商议价 |

### 3.1 换算到 元/kWh

天然气低位热值 ≈ 9.7 kWh/Nm³（标准工程取值），所以：
```
4.5 元/Nm³ ÷ 9.7 kWh/Nm³ ≈ 0.464 元/kWh
```
